Selamat datang di Multi Editor! Alat ini dirancang sebagai ruang kerja yang fleksibel untuk berbagai kebutuhan komputasi, penelitian, dan dokumentasi. Selain itu Multi Editor ini juga dirancang sebagai Laboratorium Matematika berbasis Web dan Komputasi Awan.

Komponen Editor

  • Markdown/NoteBook: Catatan dengan dukungan LaTeX.
  • MathLive: Editor persamaan interaktif.
  • SageMath (SageCell): Komputasi aljabar cloud.
  • 🤖 Bantuan AI: Terintegrasi dengan Gemini (default) dan mendukung Custom API (misalnya API key berawalan "AQ."). Silahkan anda bertanya apa saja tentang matematika atau yang lainnya. Tentunya anda harus jaga ETIKA-AI dalam bertanya! Bila anda suka, silahkan tekan ❤️.

📋 Langkah Mendapatkan API Key Gemini (Gratis)

  1. Buka situs Google AI Studio (aistudio.google.com).
  2. Klik "Sign in" dan masuk dengan akun Google Anda. Jika belum punya, buat akun gratis terlebih dahulu.
  3. Setelah masuk, cari menu "Get API key" di sisi kiri layar.
  4. Klik tombol "Create API key" (di pojok kanan atas).
  5. Pilih "Create a new project" (beri nama bebas, misal "Proyek AI Saya") atau pilih proyek yang sudah ada.
  6. Salin kode API Key yang muncul. Simpan di tempat aman karena tidak bisa dilihat lagi setelah halaman ditutup.

Catatan: Paket gratis Google memberikan kuota yang cukup untuk penggunaan pribadi (50–500 permintaan per hari). Tidak perlu kartu kredit, dan prosesnya hanya memakan waktu 2-3 menit.

🔑 Cara Menggunakan API Key di Aplikasi (WAJIB)

  1. Buka jendela "🤖 Bantuan AI" pada aplikasi ini.
  2. Pada bagian "🔑 Masukkan API Key Gemini Anda (wajib)", tempelkan API Key yang sudah Anda salin.
  3. Klik "💾 Simpan API Key". Kunci akan tersimpan di browser Anda.
  4. Sekarang fitur Bantuan AI akan langsung menggunakan API Key pribadi Anda.

Peringatan Keamanan: API Key bersifat rahasia seperti kata sandi. Jangan membagikannya kepada orang lain. Aplikasi ini menyimpannya secara lokal di browser Anda, tidak dikirim ke server manapun.

Fungsionalitas Tambahan

  • 💾 Simpan Semua: Unduh semua editor ke file JSON.
  • 📂 Import File: Muat kembali file JSON.
  • 🧹 Bersihkan Editor: Hapus semua editor.
  • ☀️/🌙 Mode Gelap: Tema terang/gelap otomatis tersimpan.

Relevansi di Era AI dan Komputasi Awan

Multi Editor ini mendukung integrasi AI dengan berbagai provider. Anda dapat menggunakan Gemini API (gratis) via proxy atau memasukkan API Key sendiri dari layanan AI apapun (OpenAI, Anthropic, Mandari, dll) dengan menyesuaikan Custom API URL dan Model.
